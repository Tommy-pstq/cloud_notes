
### Chez l'enfant 
- Série d'expérience connue :
	- Expérience du marshmallow = test de récompense différée (3 - 5 ans).
		- Récompense immédiate VS différée mais plus importante 
		- Enfant devant une assiette vide, un marshmallow est posé dedans.
		- 2 possibilités :
			- Le manger tout de suite, mais il n'en aura qu'un seul.
			- Attendre le retour de l'expérimentateur sans le manger (15 minutes).
				- Aura droit à un second marshmallow 
		- -> Petite récompense immédiate ou grosse différée 
		- Métrique :
			- Combien de temps avant de manger le marshmallow, si il est mangé.
		- Performances très variables 
			- Mangent dans les minutes / secondes après le départ : très peu de capacité d'inhibition
			- D'autres attendent jusqu'à la fin de l'expérience 
			- -> Bon indicateur des capacités de self control (inhibition)
	- Etude de la représentation des récompenses :
		- Soit visible : marshmallow visible 
			- Tps moyen = 6 minutes
			- Cherchent à détourner leur attention mais y revienne irrémédiablement. 
		- Soit caché : marshmallow caché sous un couvercle. 
			- Tps moyen = 11 minutes 
	- Accès visuel -> favorise récompense immédiate. 
	- Etude rôle de l'attention :
		- Penser à la récompense 
		- Penser à quelque chose de plaisant (une chanson de leur choix ou manipuler un jouet) = distraction.
		- -> 
			- Attente plus courte si focus sur la récompense que si situation de distraction. 
			- Donc : self control lié à la capacité de détourner son attention de l'objet désiré. 

- Rôle de l'environnement dans l'expérience du marshmallow :
	- Facteur qui influence les résultats au test : **confiance dans l'environnement**. 
	- Hypothèses :
		- Résistance à la récompense immédiate = uniquement self control
		- Résistance à la récompense immédiate : dépend aussi de la confiance des enfants dans l'environnement.
			- Confiance faible -> cèdent plus rapidement
	- Tester l'impact de la confiance dans l'adulte sur les résultats. 
		- P1 : Table avec des crayons et feuille, l'adulte promet des crayons quand il reviendra.
			- Revient effectivement avec des crayons -> confiance ++ 
			- Revient sans crayons -> Confiance --
		- P2 : Marshmallow classique.
	- Output : 
		- 10% seulement attendent si l'adulte n'est pas crédible 
		- 60% si l'adulte est crédible. 


### Chez l'animal 

- Capacité cognitive complexe = pas que chez l'homme.
	- Adaptation des méthodes expérimentales pour les animaux.
		- Pour la sèche et le perroquet.

- Perroquet :
	- Entrainé à ce que "wait" -> récompense différée. 
		- Nourriture non-attractive à disposition, attractive sous la main.
			- Accès sous la main si pas touché à la non-attractive suite au mot "wait"
				- Plusieurs fois. 
				- Mains de plus en plus longtemps.
			- Outputs : peuvent attendre jusqu'à 15 minutes pour avoir la meilleure récompense, dans la majorité des cas. 
				- Utilisation de capacité de self control (>enfants 4-5 ans.)

- Sèche 
	- Crevette >> Crabe 
	- Tous les midi : 1 crabe de sûr.
		- Si il est mangé : pas d'autres repas. 
		- S'il n'est pas mangé 
			- Le soir : 
				- La moitié reçoivent une crevette de manière certaine.
					- Récompense différée certaine. 
			- L'autre moitié : crevette 1j sur 2 
				- Récompense différée incertaine 
	- Output : 
		- Si récompense certaine : De moins en moins de crabes sont mangés le midi, jusqu'à ce que toutes attendent le soir pour avoir leur crevette.
		- Si récompense incertaine : le crabe est mangé le midi en majorité. 
		- -> Preuve de self control pour atteindre une récompense si elle est certaine. 



- Facteur d'influence :
	- Environnement 
	- Intérêt utilité 
- Variable inter-individu 
- Attention focalisée + inhibition
